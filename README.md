**ASCII Player**  
Real-time ASCII Art Converter with Webcam Support

Turn your photos, GIFs, videos, and live webcam into stylish ASCII art instantly.

**Key Features:**
- Live webcam ASCII conversion
- Support for images, GIFs, and videos
- Collapsible control panel with brightness, contrast, and width adjustment
- Colorful ASCII mode
- Edge enhancement option
- Easy saving and photo capture

Built with Python + OpenCV + Tkinter
